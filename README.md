# Fine-Tuning Pretrained CNN using ResNet50

## Description
This project demonstrates how to fine-tune a pretrained ResNet50 model
for a custom image classification task.

## Model Used
ResNet50 pretrained on ImageNet.

## Steps:
1. Load pretrained model
2. Freeze base layers
3. Add custom classifier
4. Fine-tuning last layers

## Requirements
- TensorFlow
- NumPy
- Matplotlib

## How to Run
python train.py# CNN---finetuning
